# iOS 快速运行指南

## 第一步：安装 CocoaPods 依赖

```bash
cd /Users/mac/Documents/code/LingoGo/lingo_flutter/ios
pod install
cd ../..
```

如果遇到权限问题，可能需要：
```bash
sudo gem install cocoapods
```

## 第二步：运行应用

### 方法一：使用 Flutter 命令（最简单）

```bash
cd /Users/mac/Documents/code/LingoGo/lingo_flutter
flutter run
```

Flutter 会自动：
- 检测可用的 iOS 设备或模拟器
- 如果没有运行的模拟器，会自动启动一个
- 编译并运行应用

### 方法二：使用 Xcode（适合调试）

1. 打开 Xcode 项目：
```bash
open ios/Runner.xcworkspace
```
⚠️ **重要**：必须打开 `.xcworkspace` 文件，不是 `.xcodeproj`

2. 在 Xcode 中：
   - 选择目标设备（顶部工具栏）
   - 点击运行按钮（▶️）或按 `Cmd + R`

## 常见问题

### 1. 找不到设备

```bash
# 查看可用设备
flutter devices

# 启动模拟器
open -a Simulator

# 在特定设备运行
flutter run -d <device-id>
```

### 2. CocoaPods 错误

```bash
cd ios
pod repo update
pod install --repo-update
cd ..
```

### 3. 签名错误

在 Xcode 中：
1. 选择 `Runner` target
2. 打开 `Signing & Capabilities`
3. 勾选 `Automatically manage signing`
4. 选择你的 Apple ID 作为 Team

### 4. 清理重新构建

```bash
flutter clean
flutter pub get
cd ios && pod install && cd ..
flutter run
```

## 热重载

运行后，在终端可以：
- 按 `r` - 热重载（快速刷新）
- 按 `R` - 热重启（完全重启）
- 按 `q` - 退出应用

## 下一步

运行成功后，你可以：
1. 修改 `lib/app.dart` 中的代码
2. 保存后按 `r` 查看效果
3. 开始开发你的应用功能
