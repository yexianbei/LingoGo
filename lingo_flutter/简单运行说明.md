## 关于警告

1. **UIScene 警告**：这只是 iOS 13+ 的一个警告，不影响应用运行。可以暂时忽略。

2. **mprotect 错误**：这是 Flutter 引擎在真机上的已知问题，通常是警告性的，不影响应用功能。

3. **Android embedding 警告**：这是 Android 相关的，不影响 iOS 运行。

```bash
flutter run -d 00008101-001908E23A29001E
```

或者让 Flutter 自动选择设备：
```bash

```

## 如果还有编译错误

1. **清理并重新构建**：
```bash
flutter clean
flutter pub get
cd ios
pod install
cd ..
flutter run
```

2. **检查 Xcode 项目**：
```bash
open ios/Runner.xcworkspace
```
在 Xcode 中检查是否有编译错误。
